# fB_Bot

This is a python bot that impliments threading to ensure the work is done faster

please note that all the links to be scrapped are pasted into the urlsList.txt

NB make sure that the link does not have / at the end ie

https://www.facebook.com/ProgrammersCreateLife/ is wrong and
https://www.facebook.com/ProgrammersCreateLife is correct

You should add urls to the facebookUrlList.txt file located at resources folder

saved urls are in whatsAppUrls.txt in the same folder as facebook urls


to run the application

cd to the applications directory.... penye uki ls unaona main.py

run: pip install -r requirements.txt
run: python3 main.py
